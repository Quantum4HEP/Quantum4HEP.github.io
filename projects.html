Unfolding with Quantum Annealers - QUnfold

This project introduces a quantum-based solution to the statistical unfolding problem in High-Energy Physics (HEP). The approach reformulates the problem as a Quadratic Unconstrained Binary Optimization (QUBO) model, enabling it to be addressed using Quantum Annealing (QA) on D-Wave quantum systems.

Quantum Error Mitigation with Autoencoders - QdAEnoiser

The idea behind this work is to use a particular type of Neural Network called Autoencoder, which is extensively used to reduce noise from measurements, to perform Error Mitigation of the output of a Gate-Based Quantum Computer in order to open up accurate noisy simulation for a greater number of qubits then the present state of the art.
